# Glossary — Rootstock

Full decoder ring for garden shorthand. CLAUDE.md carries the hot ~30; this is everything.

## Areas & structures
| Term | Meaning |
|------|---------|
| Bed 1 | 4'x8' raised bed, west corner next to the shed, full sun, drip irrigation installed. Primary veggie bed. |
| Bed 2 | 4'x8' raised bed, next to the garage, no water access yet. |
| the pen | 4x6 dual-chamber compost structure with raised walls, built by JET |
| the tumbler | Hexagonal rotating compost bin from Jim's Worm Farm, 2 internal sections |
| the shed | Has windows; houses the growers rack, seedling lights, heat mats. Needs power run to it. |
| the crawlspace | Dark, cool, no apparent pests — candidate root cellar / storage space for canning or root veg |
| expansion area | Yard section, currently reverted to Bermuda grass, previously tilled once before. Next-season expansion target. |
| pumpkin patch | Outside the main fence — pumpkins, watermelons, saved-seed gourd |

## Compost & soil terms
| Term | Meaning |
|------|---------|
| brewing | Active decomposition stage happening in the tumbler |
| finished compost | Fully broken-down compost, right-hand chamber of the pen |
| biochar | Lump charcoal JET is using/repurposing as biochar amendment |
| browns/greens | Standard compost carbon/nitrogen inputs — wood chips (brown) are in ample supply via Jake |
| vermiculture | Worm composting; JET's side hobby, red wigglers + earthworms from Jim's Worm Farm |
